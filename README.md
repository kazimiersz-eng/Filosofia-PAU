# Filosofia-PAU
Tabla Comparativa: Historia de la Filosofía (PAU)
Tabla web interactiva para el estudio comparativo de los principales autores de la Historia de la Filosofía (PAU/EBAU). Los datos se sincronizan en tiempo real desde una hoja de cálculo de Google Sheets, permitiendo actualizar el contenido sin modificar el código.

Características
Filtros interactivos: Permite mostrar u ocultar las columnas de autores específicos.

Secciones desplegables: Agrupa los contenidos por epígrafes temáticos que se pueden expandir o plegar.

Sin dependencias: Desarrollado con HTML, CSS y Vanilla JavaScript.

Reglas de formato en Google Sheets
Para que el código web interprete correctamente la estructura de la tabla, debes seguir estas reglas en la primera columna del Excel (Eje/Concepto):

Crear un bloque desplegable (Epígrafe): Inicia el texto de la celda con el símbolo > (ej. > A - ONTOLOGÍA) o asegúrate de que las celdas de los autores en esa fila estén vacías.

Crear un principio fundamental: Inicia el texto con una estrella ★ (ej. ★ PRINCIPIO FUNDAMENTAL) para resaltar la fila entera.

Celdas sin contenido: Deja la celda del autor vacía o escribe un guion (- o —).

Uso
Simplemente abre el archivo HTML principal en cualquier navegador web moderno.

Nota: Se requiere conexión a internet para obtener los datos. El documento de Google Sheets debe tener el acceso general configurado como "Cualquier persona con el enlace" (Lector).
